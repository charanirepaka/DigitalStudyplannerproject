 📚 Digital Study Planner
A web-based application to manage study schedules and daily tasks efficiently.

 🚀 Features
* 📅 Smart Scheduler (time-based task management)
* ✅ To-Do List (add, delete, mark complete)
* ⏰ Live clock
* 💾 Data stored using Local Storage

## 🛠️ Tech Stack
HTML | CSS | JavaScript

 ▶️ How to Run
1. Open `index.html`
2. Select Scheduler or To-Do
